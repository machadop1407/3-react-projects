## â˜ï¸ Deployment

### Deploy on Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Select the project folder (e.g., `crypto-project`, `memory-card`, or `music-player`)
5. Configure build settings:
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. Click **Deploy**

Your live application will be hosted on a custom subdomain (e.g. https://your-project.vercel.app)

### Alternative Deployment Options

- [Netlify](https://netlify.com/) - Easy deployment with continuous integration
- [GitHub Pages](https://pages.github.com/) - Free hosting for static sites
- [Render](https://render.com/) - Full-stack deployment platform
